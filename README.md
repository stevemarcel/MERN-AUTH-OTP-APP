# MERN-AUTH-OTP-APP

This is a Mern Project that focuses on Authentication. It sends OTP verification for sign up and recover password. It also stores the JWT token in a HTTP-only cookie. The technologies used in the project are React via Vite, React Redux Toolkit, React Bootstrap with a custom theme file, Express for the backend API and MongoDB for the RESTful database.
